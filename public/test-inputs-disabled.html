🧪 Test - Désactivation des Inputs

Vérification que seuls les inputs de la section active sont envoyés

📋 Instructions

1. Cliquez sur "Tester le script v3.0" pour vérifier que le nouveau script est chargé
2. Cliquez sur "Vérifier les inputs disabled" pour voir l'état des champs
3. Cliquez sur "Compter les valeurs envoyées" pour vérifier qu'il n'y a qu'une seule valeur par champ
4. Si tout est OK, allez sur le formulaire principal

🔍 Tests

✅ Résultats attendus

  • Script v3.0: Le fichier doit contenir "VERSION CORRIGÉE" et "disableSection"
  • Inputs disabled: 3 inputs avec disabled=true, 1 avec disabled=false
  • Valeurs envoyées: 1 seule valeur pour montant et currency

🐛 Si les tests échouent

  • Script pas v3.0: Videz le cache (Ctrl + Shift + Delete)
  • Tous les inputs enabled: Le script v3.0 n'est pas chargé
  • Plusieurs valeurs: Les sections cachées ne sont pas désactivées
